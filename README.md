# Agri-Tourism & Farm-Stay Website

This is a React + Vite + TailwindCSS + Firebase starter website for Agri-Tourism experiences in Ghana.  
It allows users to:

- Explore farm-stay experiences like Cocoa Cultivation Weekends and Pineapple Harvest Festivals
- View galleries of farm activities
- Book experiences via a Firebase-powered form

## Tech Stack

- React 18
- Vite
- TailwindCSS
- Firebase (Firestore)
- React Router Dom

## Pages

- **Home:** Hero section with call-to-action
- **Experiences:** Showcase of curated farm experiences
- **Booking:** Form to submit bookings to Firebase
- **Gallery:** Image gallery of farm activities

## Deployment

This project can be deployed to Vercel for a live website.

## Instructions

1. Clone or upload this repository to GitHub
2. Update `src/firebase.js` with your Firebase project config
3. Deploy to Vercel or run locally:
